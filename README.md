 # React +# نظام إدارة المتطوعين - منصة تطوع

## وصف المشروع

منصة إلكترونية لإدارة المتطوعين في الجمعيات الخيرية، تتيح تسجيل وإدارة بيانات المتطوعين، تقييماتهم، ومتابعة أنشطتهم التطوعية.

## المميزات الرئيسية

- إدارة بيانات المتطوعين الشخصية والأكاديمية
- نظام تقييمات للمتطوعين
- متابعة الشكاوى والملاحظات
- واجهة مستخدم سهلة وبسيطة
- تصميم متجاوب يعمل على جميع الأجهزة

## التقنيات المستخدمة

### Frontend

- React.js
- Tailwind CSS
- React Router
- Axios للاتصال بالخادم

### Backend

- Laravel (يجب إضافته لاحقاً)

## متطلبات التشغيل

- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

## طريقة التشغيل

1. استنساخ المشروع:

   ```bash
   git clone https://github.com/muhammedalba/volunteer_medlife.git
   cd volunteer_medlife
   ```

2. تثبيت الاعتماديات:

   ```bash
   npm install
   # أو
   yarn
   ```

3. تشغيل بيئة التطوير:
   ```bash
   npm run dev
   # أو
   yarn dev
   ```

## هيكل المشروع

```
src/
├── components/        # المكونات المشتركة
├── pages/            # صفحات التطبيق
├── services/         # خدمات API
└── contexts/         # سياقات React
```

## المساهمة

1. قم بعمل Fork للمشروع
2. أنشئ فرعاً جديداً: `git checkout -b feature/اسم-الميزة`
3. قم بحفظ التغييرات: `git commit -m 'إضافة ميزة جديدة'`
4. ادفع التغييرات: `git push origin feature/اسم-الميزة`
5. قم بفتح طلب سحب (Pull Request)
